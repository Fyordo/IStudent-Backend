# Парсинг новостей
Данные берутся с сайта http://mmcs.sfedu.ru:
 - `название новости`
 - `дата публикования новости`
 - `автор статьи`
 - `текст новости`

и группы ВК: https://vk.com/sic_mmcs:
 - `дата публикования поста`
 - `текст поста`
 - `файлы поста (фотографии, ссылки)`

Все спарсенные посты сохраняются в json файлах:
 - *data.json*
 - *data_vk.json*
